# Support

Use these channels in order:

1. Check [README.md](README.md) and the files in [docs](docs).
2. Search existing GitHub issues.
3. Open a bug report or feature request with the matching issue template.

When requesting support, include:

- Plugin name and version.
- Paper or Folia version.
- Java version.
- Other Luxorium plugins installed.
- Relevant configuration.
- Full error logs or reproduction steps.

Do not include tokens, passwords, private IP addresses, or unnecessary player data.

# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x | Yes |

## Reporting A Vulnerability

Please do not open a public issue for vulnerabilities that could compromise a
server, player data, shop inventory, or economy balances.

Report security issues through GitHub's private vulnerability reporting if it is
available for this repository. If that option is unavailable, contact the
maintainer through the profile linked on the repository owner account.

Include the affected version, server software/version, Java version, steps to
reproduce, impact assessment, and relevant logs or configuration snippets with
secrets removed.

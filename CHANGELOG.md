# Changelog

## 0.1.0 - Initial Foundation

- Added Java 25 Gradle project for Mercatus.
- Added Folia/Paper plugin descriptors.
- Added SQLite storage with WAL mode for shops and transactions.
- Added chest shop creation, ownership, listing, info, removal, and admin commands.
- Added sell-shop and buy-shop interaction foundations.
- Added optional Aureus payment bridge and no-op fallback provider.
- Added configurable shop fees, transaction tax, shop limit, blocked items, and allowed worlds.
- Added unit tests for shop validation, item keys, payment fallback, tax calculation, max shops, and blocked items.
